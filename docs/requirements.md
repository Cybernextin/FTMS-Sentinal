# Required Project Libraries

To run the SEC-OPS Sentinel platform, the following Python libraries must be installed:

- **watchdog**: Real-time file system monitoring.
- **requests**: External API communication.
- **Flask**: Core web application framework.
- **Flask-Login**: User session and authentication management.
- **Flask-Bcrypt**: Password hashing and encryption.
- **fpdf**: PDF report generation.
- **python-dotenv**: Environment variable management.
- **Flask-WTF**: CSRF security and form validation.

### Installation
You can install these dependencies using the following command:
```bash
pip install -r requirements.txt
```
